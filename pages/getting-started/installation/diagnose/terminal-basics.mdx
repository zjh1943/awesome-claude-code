# 终端基础入门教程

很多技术文档或排查步骤都需要在“终端”或“命令行”中执行。对于初学者来说，这可能看起来有点可怕（像黑客电影里的画面），但它其实只是操作电脑的另一种方式——通过文字指令而不是鼠标点击。

本文档将帮助你快速认识并打开 Windows 和 macOS 系统下的终端工具。

## 🍎 macOS 用户指南

在 macOS (苹果电脑) 上，我们主要使用 **"终端 (Terminal)"** 应用。默认使用的命令语言叫做 **Zsh** (在旧版 macOS 上是 Bash)。

**1. 如何打开终端？**

**方法一：使用聚焦搜索 (Spotlight) —— 最快！**
1.  按下键盘上的 `Command (⌘)` + `空格` 键。
2.  在搜索框中输入 `Terminal` 或 `终端`。
3.  按回车键打开。

**方法二：通过访达 (Finder)**
1.  打开 "访达 (Finder)"。
2.  进入 "应用程序 (Applications)" 文件夹。
3.  找到并打开 "实用工具 (Utilities)" 文件夹。
4.  双击 "终端 (Terminal)" 图标。

**2. 我怎么知道它打开了？**
你会看到一个白底黑字（或黑底白字）的窗口。
通常会显示你的计算机名和用户名，后面跟着一个光标。
例如：
```
MyMacBook:~ username$ 
```
或者
```
MyMacBook:~ username%
```

**3. 如何判断我用的是 Zsh 还是 Bash？**
*   **Zsh** (推荐/默认)：提示符通常以 `%` 结尾。
*   **Bash** (旧版)：提示符通常以 `$` 结尾。
*   **确认命令**：在终端里输入 `echo $SHELL` 然后按回车。
    *   如果显示 `/bin/zsh`，就是 Zsh。
    *   如果显示 `/bin/bash`，就是 Bash。

## 🪟 Windows 用户指南

Windows 上主要有两种命令行工具：
1.  **PowerShell** (蓝色背景)：功能更强大，**推荐使用**。
2.  **CMD / 命令提示符** (黑色背景)：比较老旧，功能有限。

**1. 如何打开 PowerShell (推荐)？**

**方法一：通过开始菜单搜索**
1.  点击屏幕底部的 "开始" 按钮（或按键盘上的 Windows 徽标键）。
2.  直接输入 `PowerShell`。
3.  点击 "Windows PowerShell" 打开。

**方法二：右键开始菜单 (Win 10/11)**
1.  右键点击 "开始" 按钮。
2.  选择 "Windows PowerShell" 或 "终端 (Terminal)"。

**2. 如何打开 CMD (命令提示符)？**

**方法一：运行命令**
1.  按键盘上的 `Win + R` 键，打开 "运行" 对话框。
2.  输入 `cmd`。
3.  按回车键。

**3. 如何区分 PowerShell 和 CMD？**

| 特征 | PowerShell | CMD (命令提示符) |
| :--- | :--- | :--- |
| **默认背景色** | 🟦 **深蓝色** | ⬛ **黑色** |
| **提示符前缀** | `PS C:\Users\Name>` | `C:\Users\Name>` |
| **窗口标题** | Windows PowerShell | 命令提示符 |
| **推荐程度** | ⭐⭐⭐⭐⭐ (首选) | ⭐⭐ (仅在特定老旧场景使用) |

## 🎓 新手必读：基础概念与操作

无论你是 Mac 还是 Windows 用户，以下几点能帮你避免 90% 的困惑：

**1. 什么是 "执行命令"？**
文档里说 "执行 `curl --version`"，意思是：
1.  **复制** 这行代码。
2.  **粘贴** 到你的终端窗口光标处。
3.  **按下回车键 (Enter/Return)**。
4.  等待屏幕上出现文字反馈。

**2. 复制与粘贴**
*   **macOS**: 使用 `Command (⌘) + C` 复制，`Command (⌘) + V`粘贴。
*   **Windows**: 
    *   在较新的 PowerShell/CMD 中，通常可以使用 `Ctrl + C` / `Ctrl + V`。
    *   **注意**：在某些旧版 CMD 中，`Ctrl + C` 是 "停止运行" 的意思。如果无法粘贴，尝试点击鼠标**右键**，或者点击窗口左上角图标 -> 编辑 -> 粘贴。

**3. 路径分隔符的区别**
如果你在手动输入文件路径：
*   **Windows** 使用反斜杠 `\` (例如 `C:\Users\Name\Desktop`)。
*   **macOS** 使用正斜杠 `/` (例如 `/Users/Name/Desktop`)。
*   *小技巧*：在终端里输入路径时，可以按 **Tab 键** 自动补全文件名，既快又准！

**4. 遇到问题怎么办？**
*   **Permission denied (拒绝访问)**：通常是因为权限不够。
    *   Windows: 尝试右键以 "管理员身份运行" PowerShell。
    *   macOS: 在命令前加上 `sudo ` (需要输入开机密码，输入时密码不会显示在屏幕上)。
*   **Command not found (找不到命令)**：说明你没安装这个工具，或者它不在你的系统路径 (PATH) 里。请参考我们的排查文档进行检查。
