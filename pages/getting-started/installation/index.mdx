# 安装和配置

<Callout type="info">
**安装前准备：**  
1. 确保你已经获取了 [API Key](/getting-started/get-api-key)
2. 确保你的网络环境常见的国际软件网站，比如：github.com, nodejs.org, nodesource.com, brew.sh 等。
</Callout>

开始使用 Claude Code 很简单！我们为每个操作系统提供了详细的安装指南。

每个安装指南都提供两种安装方式：
  - **一键脚本安装**：全自动安装，适合新手，5分钟快速完成
  - **手工安装**：完全掌控安装过程，适合有经验的用户
