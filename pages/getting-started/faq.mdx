# 常见问题

## 产品对比

### Q: Claude Code 与其它 Coding Agent 有何优势？

A: Claude Code 的优势体现在两方面：

1. **Claude Code 客户端**。它是 Coding Agent 的开山鼻祖，他开创了新的 Coding Agent 的产品形态，让 OpenAI 和 Gemini 都来效仿。它不是简单回答问题，而是能执行复杂目标，例如自动分析文件、生成重构方案、运行测试直至代码成功，甚至可直接提交GitHub。这种agentic操作方式让复杂多模块任务自动化且高效。
2. **Claude 大模型**。Claude 大模型在代码生成领域是全世界断档领先的，在各大评分榜上、以及程序员的使用过程中，都是独一档。

### Q: 用 Claude Code 写代码，与我让免费的 ChatGPT 对话框来写代码，有什么区别？

A: 主要有3个区别：

1. **自动化与自主执行能力**。Claude Code不仅能生成代码，还能自动运行测试、重构代码、解决错误，甚至直接提交代码到版本库，实现端到端编码任务。
2. **终端与IDE无缝集成**。Claude Code可以与主流IDE和终端深度整合，开发流程更加顺畅，代码生成、调试、提交一体化操作，而ChatGPT通常在浏览器窗口，需要频繁复制粘贴，工作流不连贯。
3. **专业支持和优化**。Claude Code持续由专业团队研发和优化，针对开发者需求设计，有丰富配置和扩展能力，能实际提升团队编码效率。而ChatGPT免费版功能有限，性能和专业性较弱。

## 使用方法

### Q: 我需要"魔法"才能使用吗？

A: **不需要**。大家只需要能连接我们的服务器就可以。我们的服务器作为连接 Claude 大模型的桥梁，已经帮你跨越了墙。

## 技术相关

### Q: 我使用的是原装的 Claude 大模型吗？

A: **是的**。我们只是提供了一个转发服务器，帮大家解决网络和支付问题。大家的请求最终还是发送到 Claude 官方的大模型的。

## 安全隐私

### Q: 我的数据安全吗？

A: 绝对安全。因为：

1. 大家的请求都是经过加密的，我们只提供代理转发给 Claude，无法获得原文。
2. Claude Code 不会为整个代码库建立索引，而是只在需要的时候使用相关的文件，故每次只有少数的几个文件会被发送到 Claude 大模型。
3. 我们 CC Club 的服务为超过 100 客户提供服务，很难从大家的请求片段中拼凑出你的源码，这件事几乎是不可能完成的。
