## 课程内容详解

在这门课程中，你将学习如何在日常开发工作流中有效使用 Claude Code。课程涵盖了从基础设置到高级功能的完整知识体系。

**在本课程中，你将学习**：

1. **安装与初始化**。学习如何安装 Claude Code，完成初始设置，并了解如何使用 CLAUDE.md 文件为项目提供持久化的上下文信息。
2. **上下文管理**。掌握如何有效地为 Claude Code 提供上下文信息，包括引用文件、提供说明以及管理对话历史。
3. **工具与权限**。了解 Claude Code 使用的各种工具，以及如何配置和管理工具权限以确保安全性。
4. **规划与思考模式**。学习使用规划模式让 Claude Code 先制定计划再执行，以及使用思考模式处理更复杂的任务。
5. **自定义 Slash Commands**。创建自定义的 slash commands 来简化常见的工作流程和重复性任务。
6. **MCP Servers 集成**。学习如何连接和使用 Model Context Protocol (MCP) servers 来扩展 Claude Code 的功能。
7. **创建 Subagents**。了解如何创建专业化的 subagents 来处理特定类型的任务。
8. **GitHub 工作流**。掌握 Claude Code 与 GitHub 的集成，自动化处理 issues、创建 pull requests 等。
9. **最佳实践与技巧**。学习使用 Claude Code 的实用技巧和最佳实践，提升开发效率。
