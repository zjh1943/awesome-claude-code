## 🔍 搜索与知识

### Brave Search

- **简介**：集成 Brave 搜索引擎，让 Claude Code 能够搜索网络信息
- **特点**：
  - 实时网络搜索
  - 隐私保护
  - 支持中文
- **安装**：
  ```bash
  npm install @anthropic/mcp-server-brave-search
  ```
- **配置**：
  ```json
  {
    "mcpServers": {
      "brave-search": {
        "command": "npx",
        "args": ["-y", "@anthropic/mcp-server-brave-search"],
        "env": {
          "BRAVE_API_KEY": "your-api-key"
        }
      }
    }
  }
  ```
- **评分**：⭐⭐⭐⭐⭐
- **维护状态**：✅ 活跃维护
- **链接**：[GitHub](https://github.com/anthropics/mcp-servers/tree/main/src/brave-search)

## 📋 使用建议

### 如何选择 MCP Server

1. **明确需求**：确定你需要什么功能
2. **查看维护状态**：优先选择活跃维护的项目
3. **阅读文档**：了解配置和使用方法
4. **安全考虑**：注意权限和 API 密钥管理

### 配置最佳实践

```json
{
  "mcpServers": {
    // 使用环境变量存储敏感信息
    "service-name": {
      "command": "npx",
      "args": ["-y", "package-name"],
      "env": {
        "API_KEY": "${SERVICE_API_KEY}"  // 从环境变量读取
      }
    }
  }
}
```

### 安全提示

<Callout type="error">
🔒 **重要**：

- 不要在配置文件中硬编码 API 密钥
- 使用环境变量管理敏感信息
- 限制文件系统访问权限
- 定期更新 MCP Server 版本
</Callout>
